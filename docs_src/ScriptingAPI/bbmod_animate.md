# bbmod_animate
`script`
```gml
bbmod_animate(model, animation_instance, anim_time)
```

## Description
Animates a Model.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| model | `array` | A Model structure. |
| animation_instance | `array` | An AnimationInstance structure. |
| anim_time | `real` | The current animation time. |
